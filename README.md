# Shiliang Shao — personal academic homepage

Static single-page site for GitHub Pages. No build step.

**Live:** https://leo-shao020104.github.io

## Layout

```
index.html              page
.nojekyll               disables Jekyll
assets/css/site.css     styles
assets/js/main.js       theme toggle + email copy
assets/images/          portrait.jpg
assets/pdf/             CV
```

## Edit

- Content: edit `index.html`.
- Colors / fonts: `:root { … }` at the top of `assets/css/site.css` (`--accent` is the cyan accent; `html.dark` mirrors it for dark mode).
- Deploy: commit and push to `main` — Pages rebuilds automatically.
